# Solidity contracts

This folder contains minimal Solidity contracts to get started:

- `GovernanceToken.sol` — ERC20 governance token using OpenZeppelin
- `DonationVault.sol` — Simple ETH vault that tracks donations per project
- `GrantRegistry.sol` — Registry of projects with metadata URIs
- `MatchingPool.sol` — Minimal matching pool skeleton

Deploy with Hardhat or Foundry. These contracts use OpenZeppelin imports.
